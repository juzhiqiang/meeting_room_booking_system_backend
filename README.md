## Description
nest 实现后端接口

## Installation  下载依赖
```bash
$ pnpm install
```

## Running the app  运行程序

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## 服务端口
```bash
 localhost:3000
```

## api文档地址

## Test

```bash
# unit tests

# e2e tests

$ pnpm run test:cov
```
